Чтобы собрать проект, нужен CMake
Все конфиги прописаны, надо только указать генератор и компилятор

cmake . -G Ninja -D CMAKE_C_COMPILER=clang

Так я компилировал у себя
Чтобы всё это собрать до кучи, нужна система сборки и компилятор. При их наличии вводится команда

cmake --build .

После этого должен появится ArkanumEngine.exe

Для сборки эксперимента, при генерации дополнительно указать -D EXPERIMENT=ON

Для компиляции необязательно всё делать через CMake, но он позволяет один раз указать параметры компиляции и линковки
Хотите компилировать руками - флаг вам в руки
